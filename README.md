# TryDjango==1.9
An introduction to the Django in python. This project is an attempt to recreate [try-django-1.9](https://github.com/codingforentrepreneurs/try-django-19), originally by [codingforentrepreneurs](https://github.com/codingforentrepreneurs).

This aims to create a simple Django blog. This covers a variety of Django basics as well as Django 1.9 specific material.
